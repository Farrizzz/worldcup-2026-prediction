# World Cup 2026 Winner Prediction

Project ini bertujuan untuk memprediksi peluang juara FIFA World Cup 2026 menggunakan Machine Learning.

Model akan mempelajari data historis pertandingan internasional untuk memprediksi hasil pertandingan antara dua negara. Hasil prediksi pertandingan kemudian akan digunakan untuk melakukan simulasi turnamen hingga mendapatkan estimasi peluang juara setiap tim.

## Tujuan Project

- Menganalisis data pertandingan sepak bola internasional.
- Membuat model prediksi hasil pertandingan.
- Menghitung peluang menang, seri, dan kalah antarnegara.
- Melakukan simulasi turnamen World Cup 2026.
- Menampilkan hasil prediksi dalam bentuk dashboard Streamlit.

## Dataset

Dataset utama yang digunakan adalah data historis pertandingan sepak bola internasional, berisi informasi seperti tanggal pertandingan, tim, skor, turnamen, lokasi, dan status tempat netral.

## Metode

- Data Cleaning
- Feature Engineering
- Machine Learning Classification
- Tournament Simulation
- Streamlit Dashboard
